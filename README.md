# Cannabis Scrape 

Consistent Python client to scrape Cannabis menus. Created for a friend.

## Get Started
```sh
git clone git@github.com:connorgiles/cannabis-scrape.git
cd cannabis-scrape
```


### Run Example
> Check out [./example.py](./example.py)
```sh
python example.py
python3 example.py
```


## Supported Providers
- Dutchie
- Buddi
